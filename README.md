# LLMapNaviManager
调用系统外部的地图应用,开启导航

iOS9之后,注意添加白名单:

  	<key>LSApplicationQueriesSchemes</key>
  	<array>
		  <string>baidumap</string>
		  <string>iosamap</string>
  	</array>
